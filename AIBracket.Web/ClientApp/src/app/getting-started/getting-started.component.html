Getting Started With AI Bracket

AIBracket is designed around writing bots to play easy to understand and competitive games.

We provide an easy to understand API for you to connect a program in your choice of language to compete against other bots.

Creating a Bot

Before you can connect a bot you must create an account and log in to access your profile.

Select the new bot + and enter the required fields to recieve a private key for your bot.

Statistics for every bot you control can be seen on your profile along with your match history.

Connecting a Bot

Connect to AIBracket.com:8000 and send the message BOT followed by your private key to get your bot started.

BOT myprivatekey54

This identifies you with the game server so we know which bot is connecting. Treat this like a password, and don’t give it to anyone. This can be found under your bot’s name on your profile.

Here is an example of TCP socket implementation in Python:

import socket

url = "AIBracket.com"
port = 8000
sock = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
sock.connect((url, port))
data = sock.recv(2048) # Recieving 2 kilobytes of data

The recommended recieve buffer size is 2KB, as some packets can exceed 1KB in length.

Game Events Documentation

Everything that happens in the game will be sent to you using an event system.

Each game has its own unique events specified by an enumeration as the first field delimeted by a space.

Events are sent to you at even intervals so sometimes there will be multiple updates.

These updates are delimeted by an *.

Simple Game Update:   1 13 11 * 2 2 5 2 False False

The highlighted blue indicates the what type of event it is. In this case, the events we recieved were events 1 and 2. See the table below for a full description of the events.

The numbers after the payload indicates payload following the event. This can be different data based on what the corresponding event is.

The yellow highlighted asterix * is the delimeter for events. There can be any number of events per update depending on the game.

Pacman Specific Documentation

For input, your bot will send "UP" "DOWN" "LEFT" and "RIGHT" strings to specify the next direction you want Pacman to go.

The API will continue to start new games upon completion until the bot disconnects.
