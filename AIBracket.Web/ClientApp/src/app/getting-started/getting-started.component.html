Getting Started With AI Bracket

AI Bracket is designed around writing bots to play simple games.

Based on what's going on in the game, you can design a simple AI to compete on a leaderboard for the highest score.

Creating a Bot

You can create your bot using any language.

Once you are ready to test your bot, create an account and log in to access your profile.

Select the new bot + and enter the required fields to recieve a private key for your bot.

Statistics for every bot any bot you own can be seen on your profile along with your match history.

Connecting a Bot

Connect to AIBracket.com:8000 and send the message BOT followed by your private key to get your bot started.

This will allow you to send and recieve data to the server about your match and your bot can process it.

The recommended recieve buffer size is 2 kilobytes.

BOT myprivatekey54

Here is an example of TCP socket implementation in Python:

import socket

url = "AIBracket.com"
port = 8000
sock = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
sock.connect((url, port))
data = sock.recv(2048) # Recieving 2 kilobytes of data

Game Events Documentation

Everything that happens in the game will be sent to you using an event system.

Each game has its own unique events specified by an enumeration as the first field delimeted by a space.

Events are sent to you at even intervals so sometimes there will be multiple updates.

These updates are delimeted by an *.

Simple Game Update:   1  13 11 * 3  25 2 10

The number before the payload indicates what type of event it is, in this case the numbers 1 and 3 represent different events.

The highlighted blue indicates the payload following the event. This can be different data based on what the corresponding event is.

The yellow highlighted asterix * is the delimeter for events. There can be any number of events per update depending on the game.

Pacman Specific Documentation

For input, your bot will send "UP" "DOWN" "LEFT" and "RIGHT" strings to specify the next direction you want Pacman to go.

The API will continue to start new games upon completion until the bot disconnects.

Here is an example of sending data through a TCP socket in Python:

import socket

url = "AIBracket.com"
port = 8000
sock = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
sock.connect((url, port))
data = sock.send(str.encode("UP")) # The string must be encoded to send
sock.close() # Disconnecting your bot will mean games are no longer created
