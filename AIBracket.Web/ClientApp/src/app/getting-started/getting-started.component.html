Getting Started With AI Bracket

AI Bracket is designed around writing bots to play simple games.
Based on what's going on in the game, you can design a simple AI to compete on a leaderboard for the highest score.

Creating a Bot

You can create a bot using any language. Once you are ready to test your bot, create an account and log in to access your profile. Select the new bot '+' and enter the required fields to recieve a private key for your bot.

Statistics for every bot any bot you own can be seen on your profile along with your match history.

Connecting a Bot

Connect to AIBracket.com:8000 and send a message BOT followed by your private key to get your bot started.
This will allow you to send and recieve data to the server about your match and your bot can process it.

***The API will require some information about your intentions, whether you are connecting for a single player game, versus game, or a tournament.




import socket
import random

def switchInput( c ):
if c == 0:
return 'LEFT'
elif c == 1:
return 'RIGHT'
elif c == 2:
return 'DOWN'
else:
return 'UP'

TCP_IP = '127.0.0.1'

TCP_PORT = 8000

BUFFER_SIZE = 1024

MESSAGE = "BOT Za2sMl0wpk6bou4H"

s = socket.socket(socket.AF_INET, socket.SOCK_STREAM)

s.connect((TCP_IP, TCP_PORT))
print("Connected")
userInput = 'LEFT'
s.send(str.encode(MESSAGE))

while userInput != 'STOP':
data = s.recv(BUFFER_SIZE)
data = data.decode('ascii')
data = data.split()
print(data)
s.send(switchInput(random.randint(0, 4)).encode('ascii'))

s.close()
